Вывод

Для разработчика, который любит разбираться в деталях, сроки сдвигаются вверх относительно «минимальной» оценки — но это оправдано: глубокое понимание каждого слоя (MediaPipe pipeline, OpenAI context management, WebSocket state machine) обеспечит более устойчивую и расширяемую систему.

Самый критичный первый шаг — не код, а привлечение квалифицированного массажиста или реабилитолога для формирования протоколов. Без этого AI-слой будет технически работать, но генерировать потенциально опасные советы.

Рекомендуется использовать Claude Code или Cursor с самого начала как инструмент для понимания, а не только для генерации — задавать вопросы «почему», читать генерируемый код внимательно и проверять каждый паттерн.
